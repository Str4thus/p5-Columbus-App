Columbus

Columbus ist ein Fahrzeug mit Modularen Aufsätzen. Das Fahrzeug und die Module können über diese App gesteuert werden.

Einführung

Benutzung: Auf der Dashboard Seite werden alle verfügbaren Module aufgelistet. Um eins zu benutzen einfach auf das angezeigte Modul klicken. Das Dashboard sowie die Einstellungen werden über das seitliche Menu erreicht.
